# CONTEXTE MAÎTRE --- PROJET DE MARQUE STREETWEAR « ARCHIVES INTERDITES »

## 1. OBJECTIF DU PROJET

Créer une marque internationale de vêtements en print-on-demand, vendue
principalement via Shopify, construite autour d'un univers cohérent de
mystères, versions alternatives de l'histoire, dossiers censurés,
cosmologies alternatives, sociétés secrètes, civilisations perdues,
phénomènes inexpliqués et réalités cachées.

Le projet ne doit pas ressembler à une simple boutique de « t-shirts
complotistes » ni à un catalogue de mèmes. L'objectif est de construire
une véritable marque streetwear premium, sombre, mystérieuse,
collectionnable et immersive.

Le modèle recherché : - zéro stock au lancement ; - production à la
demande ; - automatisation maximale ; - coûts initiaux faibles ; - vente
internationale ; - forte identité visuelle ; - contenu social
réutilisable ; - collections extensibles ; - possibilité de créer à
terme un véritable univers média et communautaire.

## 2. POSITIONNEMENT CENTRAL

La marque ne vend pas simplement des opinions ou des slogans.

Elle vend des pièces issues d'une « archive interdite ».

Positionnement conceptuel :

> Des vêtements inspirés des mystères, des versions alternatives de
> l'histoire, des dossiers censurés et des questions interdites.

Chaque vêtement doit donner l'impression d'être : - un fragment de
dossier classifié ; - une pièce d'archive ; - un document interdit
transformé en œuvre portable ; - un objet appartenant à un univers plus
vaste.

Le client n'achète pas seulement un motif. Il entre dans l'univers de la
marque et peut collectionner les dossiers.

## 3. PRINCIPES DE MARQUE

La marque doit être : - mystérieuse ; - premium ; - internationale ; -
visuellement forte ; - cohérente ; - collectionnable ; - subtile de face
; - spectaculaire de dos ; - identifiable sans devoir afficher de gros
slogans caricaturaux.

À éviter : - boutique fourre-tout sans cohérence ; - designs de mèmes
basiques ; - slogans criards sur toute la poitrine ; - esthétique
amateur ; - catalogue de centaines de produits sans système ; -
dépendance à un seul sujet comme la Terre plate.

## 4. ARCHITECTURE DES PRODUITS

Structure standard envisagée pour chaque t-shirt.

### Face avant --- Le sceau

Petit symbole discret de la collection ou du dossier.

Contenu possible : - symbole officiel ; - numéro de dossier ; - code de
collection ; - nom court de la collection ; - micro-typographie
technique.

Style : - petit ; - mystérieux ; - immédiatement reconnaissable ; - côté
cœur ou centre poitrine selon le design.

### Dos --- L'œuvre principale

Le dos est la pièce centrale et la principale raison d'achat.

Il peut contenir : - illustration principale ; - carte ; - schéma ; -
document ; - symbole ; - composition artistique ; - titre du dossier ; -
code ; - mentions de classification ; - coordonnées ; - dates ; -
fragments de texte ; - éléments techniques ou archivistiques.

Le dos doit être visuellement spectaculaire et pensé comme une affiche
ou une couverture de dossier.

### Manche --- Le code

Élément court et cohérent.

Exemples : - ARCHIVE // 001 - RESTRICTED - CLASSIFIED - FF // 001 -
symbole de collection - code du dossier

## 5. SYSTÈME DE DOSSIERS

Chaque design est un dossier numéroté.

Exemple :

FILE 001 --- THE MAP\
STATUS: DISPUTED\
ARCHIVE: WORLD MODEL\
ACCESS LEVEL: RESTRICTED

Chaque collection possède : - son nom ; - son symbole ; - son code ; -
sa typographie ou son traitement typographique ; - sa palette visuelle
; - ses règles graphiques ; - ses 10 dossiers de lancement.

Exemple de codification : - FF-001 ; - FF-002 ; - FF-003.

Ce système doit créer : - cohérence ; - rareté ; - collection ; - envie
de posséder plusieurs pièces ; - possibilité de drops ; - continuité
narrative.

## 7. VOLUME DE LANCEMENT ENVISAGÉ

Objectif actuel : - 12 collections ; - 10 t-shirts par collection ; -
120 designs potentiels.

Important : les 120 designs ne doivent pas être créés au hasard ni tous
produits simultanément sans validation.

Méthode recommandée : 1. définir le système de marque ; 2. verrouiller
le produit physique ; 3. créer les templates maîtres ; 4. définir les 10
dossiers de chaque collection ; 5. produire les concepts ; 6. tester les
visuels ; 7. publier progressivement ; 8. mesurer les clics et les
ventes ; 9. développer les collections gagnantes ; 10. éliminer ou
retravailler les thèmes faibles.

## 8. PRINT-ON-DEMAND ET PRODUCTION

### 8.1 Plateformes retenues

- Shopify : boutique principale.
- Printify : solution print-on-demand de référence au lancement.

### 8.2 Produit principal retenu provisoirement

**Bella+Canvas 3001 — Unisex Jersey Short Sleeve Tee**

Rôle :
- t-shirt principal de la marque ;
- base prioritaire pour les collections ;
- produit à utiliser pour construire les premiers templates et prototypes.

Raisons du choix :
- compatible avec l’architecture recherchée : petit sceau devant + grande œuvre au dos + code de manche ;
- large réseau de fournisseurs ;
- forte disponibilité internationale ;
- coupe moderne adaptée à une marque streetwear ;
- coton Airlume peigné et ring-spun ;
- bon compromis entre image de marque, flexibilité, prix et scalabilité.

Caractéristiques de référence à revalider avant production :
- tissu léger d’environ 142 g/m² ;
- tailles pouvant aller du XS au 5XL selon fournisseur et variante ;
- nombreuses couleurs ;
- plusieurs fournisseurs dans l’écosystème Printify.

### 8.3 Produit écarté comme base principale

**Stanley/Stella Creator 2.0**

Avantages :
- textile plus premium ;
- environ 180 g/m² ;
- coton biologique ;
- meilleure perception de matière.

Raison de l’élimination :
- l’offre étudiée ne permet pas d’exécuter de manière fiable l’architecture complète prévue avec impression avant + dos + manche.

Conclusion :
- candidat possible pour une future capsule premium si une configuration compatible apparaît ;
- ne doit pas bloquer le lancement.

### 8.4 Produit alternatif à forte marge

**Gildan 5000**

Rôle potentiel :
- gamme accessible ;
- tests de designs ;
- produits promotionnels ;
- marchés sensibles au prix.

Il ne doit pas être le produit principal tant que la marque cherche une perception premium.

### 8.5 Architecture d’impression retenue au lancement

**AVANT**
- petit sceau ou symbole ;
- impression discrète ;
- côté cœur ou centre poitrine selon la composition.

**DOS**
- grande œuvre principale ;
- zone artistique prioritaire ;
- première zone à optimiser pour la qualité visuelle.

**MANCHE**
- code du dossier ou symbole court ;
- exemples : `FF // 001`, `RESTRICTED`, `ARCHIVE // 001`.

**ÉTIQUETTE INTÉRIEURE**
- non prioritaire au lancement ;
- à ajouter uniquement si la marge et le fournisseur le permettent ;
- réservée potentiellement aux éditions premium.

Raison : chaque zone supplémentaire peut augmenter le coût de production. L’étiquette intérieure apporte moins de valeur visible que le devant, le dos ou la manche.

### 8.6 Économie cible du produit

Prix de vente standard recommandé :
- **39,90 €**

Prix possible pour certaines pièces ou éditions :
- **44,90 €**

Prix à éviter pour la configuration trois zones :
- **29,90 €**, sauf réduction temporaire ou produit simplifié.

Objectif de coût de production pour la configuration trois zones :
- viser idéalement **13 à 17 €** ;
- refuser une configuration durablement trop proche de 20–25 € avant livraison.

Important : le coût exact dépend du fournisseur, de la variante, de la taille et des zones d’impression. Le coût final doit être relevé dans le Product Creator Printify avant validation définitive.

### 8.7 Scénario économique de référence

Exemple de travail à confirmer :
- prix client : 39,90 € ;
- production trois zones : environ 15 € ;
- frais de paiement Shopify : environ 1,20 € ;
- marge avant publicité, TVA, retours et livraison : environ 23,70 €.

Ce calcul n’est pas une marge nette.

Le modèle final devra intégrer :
- TVA ;
- frais de paiement réels ;
- coût d’acquisition ;
- retours et remboursements ;
- remises ;
- livraison offerte ou subventionnée ;
- abonnement Printify éventuel ;
- coût des échantillons.

### 8.8 Stratégie géographique de production

Principe : **produire au plus près du client lorsque la qualité et la configuration restent identiques.**

Objectif :
- client USA → production USA ;
- client Europe → production Europe lorsque possible.

Bénéfices :
- délais plus courts ;
- frais de livraison plus faibles ;
- moins de risques douaniers ;
- meilleure expérience client ;
- meilleure scalabilité internationale.

Ne pas activer automatiquement un routage multi-fournisseurs sans vérifier :
- disponibilité des trois zones ;
- cohérence des tailles d’impression ;
- qualité du noir ;
- constance des couleurs ;
- qualité DTG/DTF ;
- délais ;
- coût final.

### 8.9 Technologies d’impression

Architecture envisagée :
- avant et dos : généralement DTG selon fournisseur ;
- manche : potentiellement DTF selon fournisseur.

Le DTF peut être adapté à un petit code de manche net et précis.

Avant validation, vérifier :
- technologie utilisée ;
- rendu sur textile noir ;
- finesse des petits textes ;
- résistance au lavage ;
- différences de texture entre DTG et DTF.

### 8.10 Dernière donnée obligatoire avant verrouillage définitif

La décision produit est **provisoirement verrouillée sur le Bella+Canvas 3001**, mais le fournisseur exact reste à choisir.

Test obligatoire dans le Product Creator Printify :

**Produit**
- Bella+Canvas 3001

**Variante test**
- noir ;
- taille M.

**Configuration**
- grande impression dos ;
- petit sceau devant ;
- code sur manche gauche.

Pour chaque fournisseur compatible, relever :
1. coût du t-shirt ;
2. coût de la deuxième zone ;
3. coût de la manche ;
4. coût total de production ;
5. livraison France ;
6. livraison Europe ;
7. livraison USA ;
8. délai annoncé ;
9. technologies d’impression ;
10. tailles exactes des zones ;
11. disponibilité du noir et des tailles ;
12. note et historique du fournisseur si disponibles.

Le fournisseur ne sera validé qu’après comparaison économique et commande d’un échantillon.

### 8.11 Dimensions de travail provisoires

Les dimensions exactes devront être remplacées par celles du fournisseur retenu.

**Fichier avant**
- environ 1500 × 1500 px ;
- PNG transparent ;
- 300 DPI ;
- petit symbole de 7 à 10 cm environ.

**Fichier dos**
- environ 4500 × 5400 px ;
- PNG transparent ;
- 300 DPI ;
- grande œuvre d’environ 25 à 35 cm de large selon la zone disponible.

**Fichier manche**
- environ 1200 × 1200 px ;
- PNG transparent ;
- 300 DPI ;
- code ou symbole court.

**Règle : aucun des 120 designs ne doit être finalisé pour la production avant verrouillage des dimensions exactes des trois zones.**

## 9. SYSTÈME DE PRODUCTION GRAPHIQUE

Ne pas créer 120 designs séparément sans cadre.

Créer d'abord : - 1 template maître pour le devant ; - 1 template maître
pour le dos ; - 1 template maître pour la manche.

Puis, pour chaque collection : - 1 symbole ; - 1 code ; - 1 système
typographique ; - 1 palette ; - 1 grille de composition ; - 1
bibliothèque d'éléments graphiques ; - 10 dossiers.

Objectif : - accélérer la production ; - garantir la cohérence ; -
réduire les coûts ; - permettre l'automatisation ; - faciliter la
génération assistée par IA.

## 10. UNIVERS DIGITAL LIÉ AUX VÊTEMENTS

Chaque vêtement peut contenir un QR code discret.

Le QR code peut mener vers : - une page « dossier » ; - une archive
interactive ; - une chronologie ; - des cartes ; - des images ; - des
sources ; - une vidéo ; - une expérience cachée ; - un accès réservé aux
membres.

Exemple :

Le client achète un t-shirt LOST EMPIRES sur la Tartarie.

Il scanne le QR code.

Il arrive sur : FILE LE-003 --- THE ERASED EMPIRE

La page peut contenir : - une introduction ; - une galerie ; - des
cartes ; - des documents ; - les principales affirmations autour du
sujet ; - les critiques ou contre-arguments ; - des pistes de recherche
; - des produits liés.

Important pour la crédibilité et la pérennité de la marque : présenter
les sujets controversés comme des dossiers, hypothèses, récits,
questions, documents et débats, plutôt que comme des faits certains
lorsqu'ils ne sont pas établis.

## 11. MOTEUR DE CONTENU

Chaque dossier doit devenir plusieurs contenus.

Un seul design peut produire : - 1 t-shirt ; - 1 hoodie ; - 1 page
dossier ; - 3 à 10 vidéos courtes ; - 1 carrousel ; - 1 email ; - 1 post
; - 1 visuel Pinterest ; - 1 publicité ; - 1 séquence de retargeting.

Boucle centrale :

MYSTÈRE → CONTENU → DOSSIER → PRODUIT → COMMUNAUTÉ → NOUVEAU DOSSIER

Le contenu n'est donc pas séparé du produit. Chaque collection est une
machine éditoriale.

Le marché existant contient beaucoup : - de designs isolés ; - de

L'opportunité est de construire une marque cohérente réunissant

Différenciation : - système de dossiers ; - codes de collection ; -
œuvres fortes au dos ; - symboles discrets à l'avant ; - marquage de
manche ; - pages d'archives ; - QR codes ; - storytelling ; - collection
; - drops ; - univers média.

## 14. PRIORITÉS D'EXÉCUTION

Ordre recommandé :

### Phase 1 --- Recherche et stratégie

-   cartographier les concurrents directs et indirects ;
-   identifier les meilleurs vendeurs et les esthétiques dominantes ;
-   vérifier les risques de propriété intellectuelle et de marques ;
-   identifier les thèmes à fort potentiel commercial.

### Phase 2 --- Marque

-   trouver le nom ;
-   vérifier le domaine ;
-   vérifier les réseaux sociaux ;
-   vérifier la marque ;
-   créer le logo ;
-   créer le sceau principal ;
-   définir la direction artistique.

### Phase 3 --- Produit

-   sélectionner le t-shirt Printify ;
-   sélectionner le fournisseur ;
-   calculer les coûts ;
-   fixer les zones d'impression ;
-   créer les templates exacts ;
-   commander des échantillons.

### Phase 4 --- Collections

-   finaliser les 12 collections ;
-   définir les codes ;
-   définir les symboles ;
-   écrire les 10 dossiers de chaque collection ;
-   classer les 120 concepts par potentiel.

### Phase 5 --- Création

-   produire les œuvres ;
-   adapter avant/dos/manche ;
-   créer les mockups ;
-   contrôler la lisibilité et l'imprimabilité.

### Phase 6 --- Boutique

-   construire Shopify ;
-   créer les pages collection ;
-   créer les pages dossier ;
-   connecter Printify ;
-   mettre en place le tracking ;
-   configurer emails et retargeting.

### Phase 7 --- Acquisition

-   produire les contenus courts ;
-   automatiser la publication ;
-   tester l'organique ;
-   tester les publicités sur les designs gagnants ;
-   développer les collections qui convertissent.

## 15. PROCHAINE DÉCISION À PRENDRE

Avant de produire les 120 designs, la prochaine étape prioritaire est :

> Sélectionner le produit physique exact et le fournisseur
> print-on-demand afin de verrouiller les contraintes de production.

Ensuite : 1. créer les templates maîtres exacts ; 2. définir le nom de
marque ; 3. construire l'identité ; 4. finaliser les 120 dossiers ; 5.
produire les premiers designs.

## 16. RÈGLE DIRECTRICE DU PROJET

Toujours se demander :

> Est-ce que cela ressemble à un simple t-shirt imprimé, ou à une pièce
> authentique provenant d'un univers interdit ?

Si la réponse est « simple t-shirt », le concept doit être retravaillé.

La marque doit vendre une sensation d'accès, de découverte, de
collection et d'appartenance --- pas seulement du textile.

## Les 3 fichiers par design

### DOS — l'œuvre · **4200 × 4800 px** (portrait, 300 DPI, PNG transparent)
- Composer plein cadre ; le sujet déborde jusqu'au bord.
- **Zone sûre** (titre dossier, N°, vital) : ~5 % de marge → **~3800 × 4460 px** centré.
- **Bleed** : ~1 cm par bord peut être rogné → rien de vital dedans.

### AVANT — le sceau · **1050 × 1050 px**
- Sceau vectoriel exporté à 1050², placé en haut à gauche de la zone (poitrine gauche, ~7–8 cm sous l'épaule).

### MANCHE GAUCHE — le code · **898 × 898 px** (carré)
- Code court centré (ex. `ADYTON // 001`), marge interne ~8 %.

## 3 réalités POD
1. Zone d'impression = **rectangle fixe** (ne suit pas la taille du tee).
2. **Pas de bord-à-bord** en DTG : saturer la zone max avec débord dans le bleed.
3. **Tee sombre = sous-couche blanche** → rendu un peu moins vif, risque craquelage grande impression → **échantillon obligatoire**.

## Règles de génération
- Générer directement à la taille px exacte (jamais agrandir après coup).
- PNG transparent, sRGB, effets aplatis.
- Un seul fichier de dos par design (4200×4800), valable partout.
- Palette : encre `#17130D` · papier `#EAE3D2` · rouge `#A62B22`.
